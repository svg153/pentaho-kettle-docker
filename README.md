# pentaho-kettle-docker

See kettle or spoon folder for more details.

## Reqs

* Docker

## Run

To run Pentaho Data Integration (Spoon).
Run the [`run-spoon7.sh`] script:

```shell
$ ./run-spoon7.sh
```

To see more details:

```shell
$ run-spoon7.sh -h
```
